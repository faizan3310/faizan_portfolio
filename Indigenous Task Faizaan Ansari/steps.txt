------------------------------Steps for running the project---------------------------------------

1. In the visual studio code editor, open the project folder from file open folder option.
2. All the project files will be shown on the left side of the editor.
3. open the App.js file by clicking on it.
4. Click on the terminal option at the top of the editor.
5. Now click on New Terminal. Terminal will be opened from bottom.
6. Change the project path/directory to toolbar by using cd command.
7. Enter npm start.
8. Project will be open in your default browser.
9. Now enter the keyword you want and click on SEARCH option, it will show loading and result will be displayed.
10. Click on GENERATE CITATION option than click on GENERATE, result will be displayed.
